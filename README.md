# Algebraic Math Tutor

Description

### Configuration for Use

1. Install Ollama
   https://ollama.com/download

2. Start Ollama (runs in background automatically on Windows)

3. Pull model:
   ollama run llama3

4. Install dependencies:
   npm install

5. Start server:
   node src/server.js

6. Open app:
   http://localhost:3000
